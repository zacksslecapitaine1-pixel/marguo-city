## 🚀 DÉPLOIEMENT SUR GITHUB PAGES

1. Créer un dépôt GitHub nommé `marguocity.github.io`
2. Uploader tous les fichiers du projet (y compris le dossier `images/`)
3. Aller dans **Settings → Pages → Branch: main → Save**
4. Le site sera accessible sur : `https://marguocity.github.io`

> ⚠️ Si le repo a un autre nom (ex: `hotel-marguo`), mettre à jour les URLs dans `sitemap.xml`
